# 나율이의 하브루타 🚌📚

5세 나율이를 위한 하브루타 독서 대화 앱 (PWA). 엄마가 책을 읽어주고, AI 짝꿍이 음성으로 하브루타 5단계 대화를 진행합니다. GPTs 프롬프트 「나율 전용 하브루타 GPT v1.0」을 그대로 이식했습니다.

## 주요 기능

- 한글책 / 영어책 모드 (홈 화면 큰 버튼)
- 읽기 대기 화면: 책 읽는 동안 AI가 끼어들지 않음 → [다 읽었어!] 버튼으로 대화 시작
- 음성 대화: 마이크 버튼(Web Speech STT) + AI 답변 자동 읽어주기(TTS, 문장별 한/영 음성 자동 전환)
- 영어책 모드에서 아이가 한국어로 답할 수 있게 듣기 언어 토글 제공
- 하브루타 5단계 진행도(엄마용 작은 점 표시, 아이에겐 단계 언급 안 함)
- 오늘 짧게 / 보통 / 오늘 깊게 토글
- 엄마 설정: 사고 Lv / 한글 K레벨 / 영어 E레벨 개별 조정 (프롬프트에 동적 주입)
- 대화 종료 후 엄마 피드백 + 독서 기록 자동 저장 (Supabase, 미설정 시 기기 저장)
- PWA: 안드로이드/아이폰 홈 화면에 앱으로 설치 가능

## 배포 방법 (GitHub 웹 UI + Vercel)

1. GitHub에서 새 저장소 생성 (예: `nayul-havruta`)
2. 이 zip의 압축을 풀고 **폴더 안의 파일 전체**를 GitHub 웹 UI에 드래그 앤 드롭 → Commit
3. Vercel → Add New Project → 해당 저장소 Import
4. **Environment Variables** 추가:

| 이름 | 값 | 필수 |
|---|---|---|
| `OPENAI_API_KEY` | OpenAI API 키 | ✅ 필수 |
| `OPENAI_MODEL` | 기본 `gpt-4.1-mini` (생략 가능) | 선택 |
| `SUPABASE_URL` | Supabase 프로젝트 URL | 선택 (기록 서버 저장용) |
| `SUPABASE_SERVICE_ROLE_KEY` | Supabase service_role 키 | 선택 |

5. Deploy → 완료!

Supabase를 쓰려면 `supabase.sql`을 Supabase SQL Editor에서 실행해 테이블을 먼저 만들어 주세요. 환경변수를 넣지 않아도 앱은 동작하며, 기록은 기기(localStorage)에 저장됩니다.

## 앱으로 설치하기

- **안드로이드 (크롬)**: 사이트 접속 → 메뉴(⋮) → "홈 화면에 추가" 또는 "앱 설치"
- **아이폰 (사파리)**: 사이트 접속 → 공유 버튼 → "홈 화면에 추가"

## 사용법

1. 홈에서 책 제목 입력(선택) → 한글책 또는 English Book 선택
2. 엄마가 책을 소리 내어 읽어줌 (앱은 조용히 대기)
3. [다 읽었어! 이제 얘기하자] → AI가 첫 질문을 음성으로
4. 나율이가 🎤 버튼 누르고 말하기 → 다 말하면 버튼 다시 누르기
5. 끝낼 땐 [🌙 오늘은 여기까지] → 부드러운 마무리 인사
6. [👩 엄마 피드백 + 기록 저장] → 강점/다음 목표/다음 질문 확인, 기록 자동 저장

엄마가 마이크로 "오늘은 선택형 질문 많이 해줘" 같은 지시를 말해도 즉시 반영됩니다.

## 알아두기

- 음성 인식은 안드로이드 크롬에서 가장 잘 동작합니다. 아이폰은 사파리로 열어 주세요. 미지원 브라우저에서는 키보드 입력 모드로 자동 안내됩니다.
- TTS 음성 품질은 기기 내장 음성을 사용합니다. 안드로이드는 설정 → 텍스트 읽어주기에서 Google TTS 한국어/영어 음성을 설치하면 훨씬 자연스러워요.
- AI가 모르는 책은 내용을 지어내지 않도록 프롬프트에 고정되어 있습니다. 잘 알려지지 않은 책은 대화 시작 후 엄마가 줄거리를 한두 문장 말해주면(마이크 또는 키보드) 그걸 기준으로 대화합니다.

## 파일 구조

```
app/
  page.js            홈 (모드 선택)
  session/page.js    읽기 대기 → 음성 대화 → 마무리
  parent/page.js     엄마 설정 (레벨 조정)
  records/page.js    독서 기록 목록
  api/chat/route.js    OpenAI 대화 API
  api/report/route.js  엄마 피드백 + 기록 생성 API
  api/records/route.js Supabase 기록 저장/조회 API
lib/prompt.js        하브루타 시스템 프롬프트 (GPTs v1.0 이식)
public/              PWA manifest, 아이콘, 서비스워커
supabase.sql         기록 테이블 생성 SQL
```
